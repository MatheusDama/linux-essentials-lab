Repositório dedicado ao estudo aprofundado de sistemas operacionais Linux, baseado no currículo oficial Cisco Networking Academy (NDG Linux Essentials). Aqui documento meus laboratórios práticos, scripts de automação em Bash e conceitos de administração de sistemas.

## 🚀 O que este repositório demonstra sobre minhas habilidades?
* **Administração de Sistemas:** Gerenciamento de usuários, grupos, permissões (`chmod`, `chown`) e segurança.
* **Automação:** Criação de Shell Scripts aplicados a cenários reais de infraestrutura.
* **Networking & Diagnóstico:** Uso de ferramentas de redes para troubleshooting (`ping`, `netstat`, `ifconfig/ip`).

## 📁 Estrutura dos Laboratórios
* `/labs`: Desafios propostos pelo curso resolvidos via CLI.
* `/scripts`: Automações em Bash desenvolvidas por mim para fixação do conteúdo.

## 🔧 Como Replicar os Labs
1. Clone o repositório: `[git clone https://github.com/seu-usuario/linux-essentials-lab.git](https://github.com/MatheusDama/linux-essentials-lab.git)`
2. Certifique-se de estar em um ambiente Linux (ou utilizando WSL2/Docker).
3. Para rodar os scripts de automação, dê permissão de execução: `chmod +x scripts/backup.sh`

## 🧠 Desafios & Aprendizados
* **Desafio:** Entender a fundo a máscara de permissões octal e simbólica.
* **Solução:** Criei um guia visual rápido na pasta `/labs/02-permissions` que me ajudou a fixar como o kernel interpreta os bits de leitura, escrita e execução.
